What is Bootstrap?

Bootstrap is a free and open-source html and css framework to help web-developerrs design their webage faster and easier
it's easy to use, beginners who just learned the basic of css could use bootsrap at ease by investing several days to learn and implement it.



Why Bootstrap?

Bootstrap makes front-end development faster and easier thus it increases your productivity.
It's made for folks of all skill levels, devices of all shapes, and projects of all sizes.
  • Preprocessors : Bootstraps ships with vanilla CSS, but its source code utilizes the two most popular CSS preprocessors, Less and Sass. Quickly get started with precompiled CSS or build on the source.
  • One framework, every device : Bootstrap easily and efficiently scales your websites and applications with a single code base, from phones to tablets to desktops with CSS media queries.
  • Full of features : With Bootstrap, you get extensive and beautiful documentation for common HTML elements, dozens of custom HTML and CSS components, and awesome jQuery plugins.
